Ejercicio Voluntario Nº2 - Unidad Didáctica 4

Extender el objeto Array

a) Añadir elementos al final del array

Comenzamos con un array inicial:

var array1 = [0, 1, 2];

Utilizamos el método anadir para agregar nuevos elementos:

array1.anadir(3);

El resultado será:

array1 = [0, 1, 2, 3];

b) Controlar elementos duplicados

Si no se permiten duplicados:

array1.anadir(2, false);

El resultado será:

array1 = [0, 1, 2];

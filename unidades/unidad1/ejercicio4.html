Ejercicio 4 - Unidad 1

Tabla de multiplicar

En este ejercicio, se debe ingresar un número entero positivo y mostrar la tabla de multiplicar de ese número.

a) Ingresar un número y mostrar su tabla de multiplicar


<script>
    function tablaMultiplicar() {
        let numero = prompt("Introduce un número entero positivo:");
        
        // Verificamos que el número sea positivo
        if (numero > 0) {
            let tabla = "";
            for (let i = 1; i <= 10; i++) {
                tabla += numero + " x " + i + " = " + (numero * i) + "<br>";
            }
            document.getElementById("resultado").innerHTML = tabla;
        } else {
            alert("Por favor, ingresa un número entero positivo.");
        }
    }
</script>

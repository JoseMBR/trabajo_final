Ejercicio 2 - Unidad 1

Array y Operaciones en JavaScript

En este ejercicio trabajaremos con un array que contiene distintos tipos de valores y realizaremos varias operaciones.

a) Determinar qué palabra es mayor


<script>
    let array = [true, 5, false, "hola", "adios", 2];
    let palabraMayor = array[3].length > array[4].length ? array[3] : array[4];
    alert("La palabra más larga es: " + palabraMayor);
</script>

b) Operaciones booleanas


<script>
    let resultadoTrue = array[0] || array[2];
    let resultadoFalse = array[0] && array[2];
    alert("Resultado True: " + resultadoTrue + "\nResultado False: " + resultadoFalse);
</script>

c) Operaciones matemáticas


<script>
    let suma = array[1] + array[5];
    let resta = array[1] - array[5];
    let multiplicacion = array[1] * array[5];
    let division = array[1] / array[5];
    let modulo = array[1] % array[5];
    alert("Suma: " + suma + 
          "\nResta: " + resta + 
          "\nMultiplicación: " + multiplicacion + 
          "\nDivisión: " + division + 
          "\nMódulo: " + modulo);
</script>
